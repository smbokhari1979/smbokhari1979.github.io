Primary Research Domains

My research focuses on bridging theoretical concepts with practical applications in wireless communication systems, with particular emphasis on emerging technologies and interdisciplinary approaches.

Wireless Communication

Advanced MIMO systems, 5G/6G technologies, signal detection algorithms, and wireless network optimization for next-generation communication systems.

MIMO 5G/6G Signal Detection

Signal Processing

Digital signal processing algorithms, error vector magnitude analysis, QR decomposition techniques, and lattice reduction methods for improved signal integrity.

DSP EVM QR Decomposition

IoT Systems

Internet of Things networks, smart systems design, edge computing, and IoT-enabled safety solutions including smart helmet technology.

IoT Edge Computing Smart Systems

Antenna Design

Graphene-based THz microstrip antenna arrays, quad-band antennas for biomedical applications, and advanced antenna systems for diverse reception.

Graphene THz Microstrip

Machine Learning

Machine learning algorithms for cardiovascular abnormality classification, pattern recognition in biomedical signals, and intelligent system design.

ML Classification Biomedical

Edge Detection & Multipliers

Power and area efficient multipliers for edge detection algorithms, optimizing hardware implementations for real-time signal processing applications.

VLSI FPGA Optimization

Developing novel and efficient machine learning algorithms for accurate classification of cardiovascular abnormalities using biomedical signals. The project focuses on real-time detection and monitoring systems for preventive healthcare.

  • Improve accuracy of abnormality detection

Project Overview

Designing and developing a smart helmet system integrated with IoT sensors for real-time safety monitoring and automatic accident detection. The system aims to improve rider safety through immediate emergency response.

Research Objectives:

  • Develop real-time accident detection algorithms
  • Create automated emergency alert system

Project Overview

Investigating the use of graphene materials for designing quad-band THz microstrip antenna arrays with diversity reception capabilities, specifically targeting biomedical imaging and diagnostic applications.

Research Objectives:
  • Design graphene-based antenna arrays for THz frequencies
  • Implement diversity reception techniques
  • Optimize for biomedical signal transmission
  • Characterize performance in medical imaging scenarios
